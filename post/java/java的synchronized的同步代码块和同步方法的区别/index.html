Java的synchronized的同步代码块和同步方法的区别

2021-10-09

次阅读

synchronized同步方法和同步代码块的区别

  1. 同步方法默认使用this或者当前类做为锁。

  2. 同步代码块可以选择以什么来加锁，比同步方法更精确，我们可以选择只有会在同步发生同步问题的代码加锁，而并不是整个方法。

  3. 同步方法使用synchronized修饰，而同步代码块使用synchronized(this){}修饰。

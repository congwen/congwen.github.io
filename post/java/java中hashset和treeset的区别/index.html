Java中HashSet和TreeSet的区别

2021-10-09
java
次阅读

文章目录

    • 一、实现方式
    • 二、数据是否有序
    • 三、是否可以放入null值

一、实现方式

  • HashSet：HashSet是哈希表实现的。

  • TreeSet：TreeSet是二差树实现的。

二、数据是否有序

  • HashSet：HashSet中的数据是无序的。

  • TreeSet：Treeset中的数据是自动排好序的。

三、是否可以放入null值

  • HashSet：可以放入null，但只能放入一个null。

  • TreeSet：不允许放入null值
